# BookExplorer

BookExplorer is a modern, responsive web application for discovering books, searching live data, and saving favorites.

It is built with vanilla JavaScript, HTML, and Tailwind CSS, using a real public API to fetch book information.

## Features

* Book search with live API data
* Add and remove favorites
* Persistent storage with localStorage
* Dark and light mode
* Responsive design
* Clean UI and layout
* Multi-page structure

## Tech Stack

* HTML
* Tailwind CSS
* JavaScript (ES6)
* Open Library API
* LocalStorage

## Purpose

This project was built as a professional frontend portfolio project to demonstrate real-world development skills, clean architecture, and UI/UX design thinking.
